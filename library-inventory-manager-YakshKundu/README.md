# Library Inventory Manager

Simple command-line library inventory manager built for the "Programming for Problem Solving using Python" course.

## Features
- Object-oriented Book and LibraryInventory classes
- JSON-based persistence with fault-tolerant loading
- Menu-driven CLI (Add, Issue, Return, View, Search)
- Logging and exception handling
- Packaged project structure suitable for GitHub submission

## Project structure
- `library_manager/` - package containing `book.py` and `inventory.py`
- `cli/main.py` - command-line interface
- `catalog.json` - the data file that stores the books
- `tests/` - optional unit tests
